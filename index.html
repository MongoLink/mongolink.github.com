MongoLink

An Object Document Mapper (ODM) for Java and MongoDB

GitHub » Download »

MongoLink was conceived to leave no print at all (or almost) on your project : no annotations, no super class to inherit. Mapping is declared through a fluent api directly in Java, and the configuration to get started is just a few lines long.

Install

You can either build the project yourself, using maven and our little ruby script to build each projets ; or use our maven repository , provided by cloudbees.

MongoLink also provides a set of tools to ease unit testing. Mainly, it provides an in memory implementation of MongoDB's drivers, as long with with a jUnit rule to setup your tests easily.
